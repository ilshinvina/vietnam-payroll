# 급여 규정 탭 - 베트남 세법 표시 추가 완료

**수정 일시**: 2025-11-17
**수정 파일**: settings.html (급여 규정 탭)
**목적**: 코드에 반영된 베트남 세법 규정을 사용자가 볼 수 있도록 명확히 표시

---

## 🎯 **수정 이유**

사용자 요청:
> "이것이 가능해야지 회사설정이나 급여규정에 방금의 로직의 내용을 볼수 있게........"

**문제점**:
- salary-calculator.js에서는 베트남 세법 준수 로직 적용됨
- 하지만 settings.html의 "급여 규정" 탭에는 표시되지 않음
- 사용자가 어떤 규칙이 적용되는지 알 수 없음

---

## ✅ **수정 내용**

### 1. **식대 비과세 한도 표시** (Line 870-874)

**수정 전**:
```html
<div class="info-box" style="border-color: #ff9800;">
    💡 식대는 100% 비과세입니다.
</div>
```

**수정 후**:
```html
<div class="info-box" style="border-color: #ff9800;">
    ⚖️ <strong>베트남 세법 (Decree 954/2020/NĐ-CP)</strong><br>
    💡 식대 비과세 한도: <strong style="color: #ff9800;">월 730,000 VND</strong><br>
    ※ 초과분은 과세 대상 소득에 포함됩니다.
</div>
```

---

### 2. **시급 계산 방식 추가** (새 섹션 - Line 886-902)

**추가 내용**:
```html
<!-- 4. 시급 계산 방식 -->
<div class="payroll-card">
    <h3 style="color: #9c27b0;">🧮 4. 시급 계산 방식</h3>
    <div style="margin-left: 20px; color: #666;">
        <div class="info-box" style="border-color: #9c27b0;">
            ⚖️ <strong>베트남 노동법 (Labor Code No. 45/2019/QH14)</strong><br>
            <p style="margin: 10px 0; font-size: 1.2em; color: #333;">
                <strong>시급</strong> = <strong>기본급</strong> ÷ (<strong>월 평일 수</strong> × <strong>8시간</strong>)
            </p>
            <p style="margin-top: 10px; color: #666;">
                💡 월별 평일 수는 공휴일과 일요일을 제외한 실제 근무 가능 일수입니다.<br>
                예시: 4월 (평일 22일) → 7,000,000đ ÷ (22×8) = 39,772đ/시간<br>
                예시: 5월 (평일 26일) → 7,000,000đ ÷ (26×8) = 33,653đ/시간
            </p>
        </div>
    </div>
</div>
```

**효과**:
- 시급이 매달 다른 이유를 명확히 설명
- 베트남 노동법 조항 명시
- 구체적인 계산 예시 제공

---

### 3. **초과근무 수당 비과세 규정 상세화** (Line 904-933)

**수정 전**:
```html
▪ 평일 연장근무(잔업): 시급 × 1.5배
※ 소득세: 1/3 비과세, 2/3 과세

▪ 야간근무 추가: 시급 × 1.3배 (130%)
※ 소득세: 전액 과세  ❌ 틀림!

▪ 일요일 특근: 시급 × 2.0배
※ 소득세: 1/2 비과세, 1/2 과세
```

**수정 후**:
```html
▪ 평일 연장근무(잔업): 시급 × 1.5배 (150%)
✅ 소득세: 50% 가산분 중 1/3 비과세, 2/3 과세
※ Circular 111/2013/TT-BTC 준수

▪ 야간근무 추가 (22:00~06:00): 시급 × 1.3배 (130%)
✅ 소득세: 30% 가산분 비과세, 기본 100% 과세
※ Circular 111/2013/TT-BTC 준수

▪ 일요일 특근: 시급 × 2.0배 (200%)
✅ 소득세: 100% 가산분 비과세, 기본 100% 과세
※ Circular 111/2013/TT-BTC 준수

💡 비과세 요약
- 연장근무: 50% 가산 → 1/3 비과세 (약 16.7%)
- 야간근무: 30% 가산 → 30% 비과세 (약 23.1%)
- 일요일: 100% 가산 → 50% 비과세
```

**효과**:
- 야간근무 비과세 규정 정확히 표시
- 베트남 세법 조항 명시 (Circular 111/2013/TT-BTC)
- 비과세 요약으로 한눈에 이해 가능
