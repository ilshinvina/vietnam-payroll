## 📊 **급여 대장 출력 검증** (index.html)

### 대시보드 확인
- [ ] 이번 달 총 급여가 정확한가?
- [ ] 확정 인원 진행률이 정확한가? (예: 3/16)
- [ ] 월별 급여 현황 테이블이 정확한가?

### 📋 상세 버튼
- [ ] 급여 대장 모달이 열리는가?
- [ ] 모든 직원이 표시되는가?
- [ ] 금액 합계가 정확한가?

### 📄 PDF 다운로드
- [ ] PDF가 다운로드되는가?
- [ ] 회사명이 올바르게 표시되는가?
- [ ] 한글/베트남어가 깨지지 않는가?
- [ ] 숫자 포맷이 올바른가? (1,000,000đ)
- [ ] 모든 열이 잘 보이는가?

### 📊 엑셀 다운로드
- [ ] 엑셀 파일이 다운로드되는가?
- [ ] 엑셀에서 열렸을 때 데이터가 정확한가?
- [ ] 한글/베트남어가 깨지지 않는가?
- [ ] 숫자가 숫자 형식으로 인식되는가?
