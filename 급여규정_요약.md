## 🧮 4. 시급 계산 방식

⚖️ **베트남 노동법 (Labor Code No. 45/2019/QH14)**

### 공식:
```
시급 = 기본급 ÷ (월 평일 수 × 8시간)
```

### 예시:
- **4월 (평일 22일)**: 7,000,000đ ÷ (22 × 8) = **39,772đ/시간**
- **5월 (평일 26일)**: 7,000,000đ ÷ (26 × 8) = **33,653đ/시간**

💡 월별 평일 수는 공휴일과 일요일을 제외한 실제 근무 가능 일수
